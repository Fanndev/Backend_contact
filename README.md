
# BACKEND CONTACT

Rest api for contact app flutter


## Run Locally

Clone the project

```bash
  git clone https://github.com/Fanndev/Backend_contact.git
```

Go to the project directory

```bash
  cd Backend_contact
```

Install dependencies

```bash
  npm install / npm i
```

Start the server

```bash
  npm run dev
```
